

# 📑 技术文档：AI 视频自动再加工流水线 (Antigravity 版)

## 1. 项目概述

本系统利用 **Antigravity** 引擎驱动一个专项 **VideoShortsAgent**。该 Agent 接收长视频，自动提取金句，并使用 **Remotion** 渲染出带动态字幕的高质量短视频切片。

## 4. 核心技能实现 (Skill Classes)

### 4.1 TranscribeSkill (语音转文字)

**逻辑**：调用 Whisper 模型，生成带时间戳的 JSON。

```python
import os
from faster_whisper import WhisperModel
import json

class TranscribeSkill:
    def __init__(self):
        self.model = WhisperModel("base", device="cpu", compute_type="int8")

    def execute(self, video_path, output_dir):
        print(f"正在转录: {video_path}")
        segments, _ = self.model.transcribe(video_path)
        
        transcript = []
        for s in segments:
            transcript.append({"start": s.start, "end": s.end, "text": s.text})
            
        dest = os.path.join(output_dir, "transcript.json")
        with open(dest, "w", encoding="utf-8") as f:
            json.dump(transcript, f, ensure_ascii=False)
        return dest

```

### 4.2 AnalysisSkill (Qwen 金句抽取)

**逻辑**：将文字喂给 Qwen-Max，返回结构化的剪辑方案。

```python
from openai import OpenAI
import json

class AnalysisSkill:
    def __init__(self, api_key):
        self.client = OpenAI(
            api_key=api_key,
            base_url="https://dashscope.aliyuncs.com/compatible-mode/v1"
        )

    def execute(self, transcript_path):
        with open(transcript_path, "r") as f:
            content = f.read()
            
        prompt = f"你是一个爆款专家。请从以下文本中选出一段15-30秒的金句，返回JSON格式，包含字段：start, end, hook_text。\n文本：{content}"
        
        response = self.client.chat.completions.create(
            model="qwen-max",
            messages=[{"role": "user", "content": prompt}],
            response_format={"type": "json_object"}
        )
        return json.loads(response.choices[0].message.content)

```

## 5. Agent 逻辑配置 (Antigravity Core)

### 5.1 `SOUL.md` (Agent 设定)

```markdown
# Role: 爆款短视频剪辑大师

## Workflow:
1. 当收到视频文件时，分配一个唯一的 `TASK_ID`。
2. 调用 `TranscribeSkill` 获取文本。
3. 调用 `AnalysisSkill` (Qwen) 确定最有爆发力的 20 秒。
4. 调用 `RenderSkill` (Remotion) 生成 MP4。
5. 任务完成后，回复用户视频，并调用 `ClearSession` 确保无记忆状态。

## Constraints:
- 严禁混淆不同任务的上下文。
- 视频字幕必须使用亮黄色，大字体。

```

## 6. 视觉模版逻辑 (Remotion/React)

在 `/remotion-project/src/Composition.tsx` 中定义字幕动画：

```tsx
import { spring, useCurrentFrame, useVideoConfig, interpolate, AbsoluteFill } from 'remotion';

export const Caption = ({ text }: { text: string }) => {
  const frame = useCurrentFrame();
  const { fps } = useVideoConfig();

  const scale = spring({
    frame,
    fps,
    config: { stiffness: 100 },
  });

  return (
    <AbsoluteFill style={{justifyContent: 'center', alignItems: 'center', bottom: 100}}>
      <div style={{
        fontSize: 80, 
        color: '#FFD700', 
        transform: `scale(${scale})`,
        fontWeight: 'bold'
      }}>
        {text}
      </div>
    </AbsoluteFill>
  );
};

```

## 7. 部署与迁移指南 (Cloud Deployment)

### 7.1 Docker 化建议

由于 Remotion 依赖浏览器，请使用以下基础镜像：
`FROM ghcr.io/remotion-dev/template-python-docker`
该镜像预装了 Python、Node.js 和渲染所需的 Chrome 依赖。

### 7.2 无状态化保证 (Anti-Confusion)

1. **任务文件夹隔离**：每个任务使用 `uuid.uuid4()` 创建独立目录。
2. **环境变量隔离**：敏感 Key 通过 `.env` 注入，而非硬编码。
